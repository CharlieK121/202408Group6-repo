# Scholar Path
Scholar Path is a web based application that allows students to explore the campus map and resources.

# Install instructions
Required programming language: Javascript
Required markup language: HTML5, CSS

Note: Visual Studio Code includes built-in JavaScript IntelliSense.

# Software Engineeering
For Scholar Path, we use software engineering principles such as agile development and user requirements based features.
We first took into account the use cases for the program and how they relate to the user, such as students and staff. From there,
we drafted a roadmap highlighting the main epic features that would provide the most value to the system. Product roadmaps are used to keep track of what will be implemented and team backlog is used to keep track of the approved features. Here, in the team backlog, we also breakdown the feature into user stories and plan for their development during our one week sprint review sessions.
